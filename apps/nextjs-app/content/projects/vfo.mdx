## What is vfo?

A utility for batch-encoding your video portfolio. Instead of manual queues, vfo analyzes each file’s resolution, codec, and color space, and makes smart decisions.

## Installation

## Option 1: macOS Installer (.pkg)

1. Download \`vfo_macOS_installer-v*.pkg\` from the **Assets** section on the GitHub Releases page.
2. **Shift-click → Open** the installer to bypass Gatekeeper (required until v1.0.0).
3. Follow the on-screen instructions.
4. Once installation completes, open Terminal and run:
   \`\`\`bash
   vfo -v
   \`\`\`
   If you see a version number, vfo was installed successfully.

## Option 2: Static ZIP

1. Download \`static_build.zip\` from the **Assets** section on the GitHub Releases page.
2. Unzip the archive.
3. Move **both** the \`vfo\` binary and \`vfo_config.conf\` to a directory in your \`$PATH\` (e.g. \`/usr/local/bin\`):
   \`\`\`bash
   sudo cp vfo vfo_config.conf /usr/local/bin/
   sudo chmod +x /usr/local/bin/vfo
   \`\`\`
4. Verify installation:
   \`\`\`bash
   vfo -v
   \`\`\`

## Installation

1. Download the latest release from GitHub.
2. Place the \`vfo\` binary in your \`$PATH\` (e.g., \`/usr/local/bin\`).
3. Ensure \`ffmpeg\` is installed and accessible.

## Usage

Run \`vfo --help\` to list all commands and options.

## Description

vfo automates batch video encoding by scanning each file’s metadata and applying tailored \`ffmpeg\` commands based on user-defined aliases and scenarios.

## Examples

- Encode all videos in a folder:
  \`\`\`bash
  vfo all_aliases
  \`\`\`
- Encode a single alias:
  \`\`\`bash
  vfo alias queen
  \`\`\`

## Detailed Options

Refer to \`vfo_config.conf\` for global settings (paths, tests, strategies) and per-alias scenarios.`}

## Why It Matters

Manual encoding of diverse video files is time-consuming and error-prone. vfo automates per-file decisions while giving you full control, saving time and ensuring consistency across your portfolio.
